User Requirements

1. Users should report all faults by email to EUL.Systems.Support@ed.ac.uk or via Sysweb

- Users are expected to have tried the basic troubleshooting notes contained in the Systems Support manual before submitting a request for assistance
- Users should ensure they can replicate the fault before reporting it to helpdesk
- Users should consult the Systems Support manual to ensure that all mandatory information is included
in the fault report

2. Users should send all developmental requests to systems.development@ed.ac.uk.

Users should include:

- details of the enhancement or development required
- estimates of any additional hardware or software needed
- dates by which the enhancement should be operational
