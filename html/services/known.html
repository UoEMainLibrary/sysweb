Common Desktop

Known Problems

Slowness of Microsoft Applications

Description: There have been reports of slowness in Microsoft Applications such as Word or Excel. The problems occur especially when the user first opens a file and it can take up to 10 seconds before editing can commence. On other occasions we have heard that typing is slow to come up when keys are pressed.

Solution: This problem is caused by a fault on the library server (lib-srv4). However, the server is due to be replaced in the near future. We have received the new equipment and MFT are currently configuring it for use. When the new server is installed, user should no longer have the current problem.

JPG and GIF not Opening an Application

Description: When the user clicks on a JPG or GIF file, an error message informs them that there is no application associated with the file. Normally, clicking on this type of file would open up a viewer such as Explorer to view the file or a image manipulation application such as Paintshop or Photoshop.

Solution: We intend fixing this problem in the next Maintenance Level (005). The default application to be launched will be Photoshop 6.0

Hanging Shutdowns

Description: PC hangs upon user shutdown at the Windows 98 splash screen.

Solution: This is a Windows 98 bug. We have downloaded a patch from Microsoft which is supposed to fix the problem. The patch was applied to all library PCs with Maintenance Level 003. However, we have had a few reports that some machines are still carrying the fault. We are investigating this with Microsoft, however user are advised that this shouldn't cause any problems with the machine and you should switch off as normal when the Windows 98 splash screen appears.
