What is a Development


Requests sent to the development team, should not be faults with the current service, instead, they should fall into one of the following categories:

  • Enhancement to existing service.

  • Request for new service.

  • Request for information.

  • Request for equipment.

  • Systems policy information.


Please make sure all faults are referred to the systems helpdesk.
